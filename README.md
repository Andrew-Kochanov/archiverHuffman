### Цель кейса
Создать консольное приложение для сжатия и восстановления файлов без потерь на основе алгоритма Хаффмана:
- Анализ частот байтов исходного файла.
- Построение дерева Хаффмана и генерация префиксных кодов.
- Упаковка данных в битовый поток с сохранением заголовка (частот символов).
- Распаковка с восстановлением исходных данных.
- Экспериментальная оценка эффективности сжатия и производительности на различных типах файлов.
